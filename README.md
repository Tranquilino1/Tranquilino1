### 🚀 Proyectos Destacados

* 📱 **[EGMobileMoneyAsist-AI-](https://github.com/Tranquilino1/EGMobileMoneyAsist-AI-)**
  Asistente de finanzas personales para Android optimizado con IA (Google Gemini 1.5 Flash). Desarrollado para ayudar a usuarios en Guinea Ecuatorial en el Hackathon 24H de FP Salesianos.

* 🌾 **[AGROTEC-GUINEA](https://github.com/Tranquilino1/AGROTEC-GUINEA)**
  Sistema web de diagnóstico agrícola potenciado por modelos de lenguaje en la nube (Groq LPU + Google Gemini API) creado para el II Foro Nacional de IA en AAUCA 2026.

* 🏥 **[SistemaClinicoAauca](https://github.com/Tranquilino1/SistemaClinicoAauca)**
  Software premium de administración y gestión clínica para centros médicos. Cuenta con automatización de expedientes de pacientes, prescripciones médicas y descargas en versión portable.

* 🏋️‍♂️ **[AAUCAGYM](https://github.com/Tranquilino1/AAUCAGYM)**
  Sistema de administración de gimnasio universitario desarrollado utilizando Spring Boot en backend y Angular para la interfaz de usuario.
